# JSON Server

## Requirment
- Node.js 
- npm أو yarn

## Steps
npm install

npx json-server --watch db.json --port 3001

npm run dev

## Catch the system

- http://localhost:8080
- JSON Server API: http://localhost:3001
- admin system : http://localhost:8080/admin
- password**root**

## API Endpoints

- GET /bookings
- POST /bookings
- PUT /bookings/:id 
- DELETE /bookings/:id
